$Revision$
Author:
Keith Visco

The class {@link org.exolab.castor.builder.SourceGenerator} is used for creating source code from an XML Schema. Currently the version of XML Schema that is supported by the SourceGenerator is a subset of the W3C XML Schema Working Draft 20000407. More information may be obtained in the Castor documentation.

The file org.exolab.castor.builder.castorbuilder.properties is used for configuring some advanced options.
